# CLI-Snake-Game
 A CLI Snake Game written in C

---

## Compilation / Installation
Linux / macOS:
```
gcc -O2 snake.c -Wall -o snake
sudo mv snake /usr/bin
```

---

## Parameters
| Parameter  | Description |
| ------------- | ------------- |
| -help, -h  | Display help information and commands |
| --version, -v  | Show version information |
| --infinity  | Infinity game mode |
| --no-initial-apple  | Do not add apple on game start |
| --no-apple-on-eat  | Do not add apple on eat |
| --sizeX <int>  | Size of table |
| --sizeY <int>  | Size of table |
| --resolution <int>  | Resolution of each cell |
| --time-apple <int>  | Milliseconds betwen each apple add (0-disabled) |
| --time-move <int>  | Milliseconds betwen each movement (0-disabled) |
